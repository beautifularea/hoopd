# [hoopd](https://github.com/zTgx/hoopd)
A simple HTTP server.

### Usage
##### Step1
```shell
git clone --recursive https://github.com/zTgx/hoopd.git
```
##### Step2
```shell
mkdir build && cd build
```
##### Step3
```shell
cmake .. && make
```
##### Step4
```shell
./bin/hoopd
```
