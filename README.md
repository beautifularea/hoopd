# [hoopServer](https://github.com/zTgx/hoopServer)
A HTTP server.

### Usage
##### Step1
```shell
git clone --recursive https://github.com/zTgx/hoopServer.git
```
##### Step2
```shell
mkdir build && cd build
```
##### Step3
```shell
cmake .. && make
```

##### Step4
```shell
make test
```

##### Step5
```shell
./bin/hoopServer
```
